🚀 Java Web Framework

Welcome! This server supports REST services and static files.

Available REST Endpoints

GET /App/hello?name=Pedro
Try it → — Returns a greeting using the name query parameter.
GET /App/pi
Try it → — Returns the value of π (Pi).
GET /App/euler
Try it → — Returns the value of e (Euler's number).

Static Files

Static files are served from target/classes/webroot.

Any unrecognized path returns Hello World!
